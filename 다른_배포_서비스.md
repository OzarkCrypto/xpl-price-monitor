# 🌐 다른 배포 서비스 옵션

Render 말고도 사용할 수 있는 다양한 배포 서비스들입니다.

## 1. Railway (가장 간단) ⭐⭐⭐⭐⭐

### 장점
- ✅ 설정 파일 자동 인식 (railway.json)
- ✅ GitHub 연동 자동 배포
- ✅ 무료 크레딧 제공
- ✅ 슬립 모드 없음 (항상 실행)

### 배포 방법
1. **https://railway.app** 접속
2. **"Start a New Project"** 클릭
3. **"Deploy from GitHub repo"** 선택
4. 저장소 선택: `OzarkCrypto/xpl-price-monitor`
5. **자동 배포!** (설정 파일 자동 인식)

### 비용
- 무료 크레딧 제공 (월 $5 크레딧)
- 사용량에 따라 과금

---

## 2. Fly.io (글로벌 배포) ⭐⭐⭐⭐

### 장점
- ✅ 전 세계 여러 지역에 배포 가능
- ✅ 빠른 응답 속도
- ✅ 무료 플랜 제공
- ✅ 커스텀 도메인 지원

### 배포 방법
1. **https://fly.io** 접속 및 가입
2. Fly CLI 설치:
   ```bash
   curl -L https://fly.io/install.sh | sh
   ```
3. 로그인:
   ```bash
   fly auth login
   ```
4. 배포:
   ```bash
   fly launch
   fly deploy
   ```

### 비용
- 무료 플랜: 3개의 공유 CPU VM
- 사용량에 따라 과금
